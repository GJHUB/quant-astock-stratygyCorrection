一、 核心量化逻辑：多头阶梯博弈 (Step-Up Oscillation)
本策略专为 A 股 120 只高波科技股池设计，摒弃突破追高，专注于在多头趋势中的“缩量极端负乖离”处进行左侧低吸与右侧确认，并在“放量极端正乖离”处逢高派发。

1. 信号触发数学表达
定义 $P_t$ 为 $t$ 日收盘价，$V_t$ 为 $t$ 日成交量。

底部低吸买点 ($S_{buy} = 1$)：需同时满足三大条件。
Beta 护航：$SMA(P, 60)t > SMA(P, 60){t-5}$
情绪极值 (负乖离)：$BIAS_t = \frac{P_t - SMA(P, 20)}{SMA(P, 20)} \times 100 < -\theta_{buy}$
抛压枯竭 (缩量)：$V_t < \alpha_{vol} \times SMA(V, 10)$
(可选防伪添加) 右侧确认：$P_t > P_{t-1}$ （即当日收盘价高于昨日，确认止跌）
顶部止盈卖点 ($S_{sell} = 1$)：满足以下其一。
情绪高潮 (正乖离)：$BIAS_t > \theta_{sell}$
量价背离：$V_t > 2.0 \times SMA(V, 10)$ 且上影线比例 $> 0.6$
无条件止损 ($S_{stop} = 1$)：
破位止损：$P_t < SMA(P, 60) \times 0.95$
二、 动态仓位管理模块：ATR 风险平价
核心思想：波动越大的股票，买的股数越少；波动越小的股票，买的股数越多。确保单次交易所承担的绝对风险值恒定。

1. 计算真实波动率 (ATR)：
$$ATR_t = \max(H_t - L_t, |H_t - C_{t-1}|, |L_t - C_{t-1}|)$$
取 14 日平滑平均：$SMA(ATR, 14)$
2. 计算单笔目标头寸 (Position Sizing)：
假设总账户资金为 $TotalCapital$，单笔交易愿意承担的最大总资金回撤风险敞口为 $R$（例如总资金的 1%）。
$$Shares = \frac{TotalCapital \times R}{SMA(ATR, 14) \times Multiplier}$$

A 股必须以 100 股（一手）为单位交易。AI Agent 在代码实现时必须对 $Shares$ 进行向下取整处理。

import pandas as pd
import numpy as np
import math

def generate_signals_and_positions(df, total_capital=1000000, risk_per_trade=0.01):
    """
    输入 df 需为单只股票的日线数据，包含: ['open', 'high', 'low', 'close', 'volume']
    risk_per_trade: 单笔交易愿意承担的账户总资金风险比例 (默认 1%)
    """
    # 1. 计算均线与乖离率 (Bias)
    df['SMA_20'] = df['close'].rolling(window=20).mean()
    df['SMA_60'] = df['close'].rolling(window=60).mean()
    df['BIAS_20'] = (df['close'] - df['SMA_20']) / df['SMA_20'] * 100
    df['Vol_SMA_10'] = df['volume'].rolling(window=10).mean()
    
    # 2. 计算 ATR (Average True Range)
    df['prev_close'] = df['close'].shift(1)
    df['TR'] = df[['high', 'prev_close']].max(axis=1) - df[['low', 'prev_close']].min(axis=1)
    df['ATR_14'] = df['TR'].rolling(window=14).mean()
    
    # 3. 识别长上影线
    df['Upper_Shadow'] = df['high'] - df[['close', 'open']].max(axis=1)
    df['Body_Total'] = df['high'] - df['low'] + 1e-8
    df['Upper_Shadow_Ratio'] = df['Upper_Shadow'] / df['Body_Total']
    
    # 4. 生成买卖信号 (布尔值)
    buy_condition = (
        (df['SMA_60'] > df['SMA_60'].shift(5)) & 
        (df['BIAS_20'] < -8.0) & 
        (df['volume'] < 0.6 * df['Vol_SMA_10']) &
        (df['close'] > df['open']) # 右侧确认：收阳线
    )
    
    sell_condition = (
        (df['BIAS_20'] > 15.0) | 
        ((df['volume'] > 2.0 * df['Vol_SMA_10']) & (df['Upper_Shadow_Ratio'] > 0.6)) |
        (df['close'] < df['SMA_60'] * 0.95) # 止损
    )
    
    df['Signal'] = 0
    df.loc[buy_condition, 'Signal'] = 1
    df.loc[sell_condition, 'Signal'] = -1
    
    # 5. 动态仓位计算 (仅在产生买入信号时计算)
    # 单笔风险金额 = 100万 * 1% = 1万。除以 ATR 得到应买股数，再按 100股(一手) 向下取整
    df['Target_Shares'] = 0
    df.loc[df['Signal'] == 1, 'Target_Shares'] = (
        (total_capital * risk_per_trade) / df['ATR_14']
    ) // 100 * 100 
    
    return df
