4. 模块3: 量化策略 (signal_generator.py) - 核心增强

目的： 修复0交易，增强捕捉洗盘买点（BIAS + RSI + MACD）。

修改点： 放宽默认阈值（theta_buy=3-5%）；T+1 shift。

输入： 过滤股、日K数据、参数。
输出： DF with 'Signal' (1买/-1卖, T+1 shift), 'Target_Shares'。

规则（更新）： 买入：SMA60上 + BIAS<-theta_buy + Vol<alpha_vol*SMA10 + RSI<rsi_thresh + HIST转正 + 阳线。

关键公式（LaTeX）：

BIAS: ( BIAS_t = \frac{P_t - SMA(P, 20)_t}{SMA(P, 20)_t} \times 100 )
RSI: ( RSI_t = 100 - \frac{100}{1 + RS_t} )
HIST: ( HIST_t = (EMA_{12} - EMA_{26}) - EMA_9(MACD) )
Shares: ( Shares = \left\lfloor \frac{Capital \times R}{ATR_{14} \times (1 + Cost)} \right\rfloor \div 100 \times 100 ), Cost=0.0035

代码patch模板：

python
复制
import talib

def generate_signals(df, theta_buy=4.0, rsi_thresh=35, alpha_vol=0.6, ...):
    # 原计算...
    buy_cond = ((df['SMA60'] > df['SMA60'].shift(5)) &
                (df['BIAS20'] < -theta_buy) &  # 放宽
                (df['volume'] < alpha_vol * df['Vol_SMA10']) &
                (df['RSI14'] < rsi_thresh) &  # 放宽至35
                (df['HIST'] > df['HIST'].shift(1)) &
                (df['close'] > df['open']))
    # 卖出类似...
    df['Signal'] = 0
    df.loc[buy_cond, 'Signal'] = 1
    df['Signal'] = df['Signal'].shift(1).fillna(0)  # T+1
    # 仓位计算...
    return df
