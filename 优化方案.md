问题诊断

回测仅产生1个买点的根因在于信号生成逻辑的概率乘积效应。详细拆解如下：

信号条件过于严苛（主因）
原逻辑采用纯AND多条件（6个指标必须同时满足），在A股高波动环境下联合概率极低（<5%）。例如，“MACD HIST转正 + 阳线 + 缩量”三者同日出现的稀有性，导致整体信号稀疏。
原信号生成公式（LaTeX）：
$$
S_{\text{buy},t} = \mathbb{I}(\text{SMA60 up})_t \times \mathbb{I}(\text{BIAS20}t < \theta{\text{buy}}) \times \mathbb{I}(\text{Vol}t < \alpha{\text{vol}} \cdot \text{SMA10Vol}_t) \times \mathbb{I}(\text{RSI14}t < \text{rsi_thresh}) \times \mathbb{I}(\text{MACD HIST}_t > 0 \land \text{MACD HIST}{t-1} \leq 0) \times \mathbb{I}(\text{Close}_t > \text{Open}_t)
$$

遗传算法优化副作用
原适应度函数（Calmar比率）未充分惩罚“交易次数过少”，导致参数收敛到极端阈值（如θ_buy=-12%），进一步压缩信号空间。v3.1日志中常见“低交易量”问题。

微观执行层过滤（放大效应）
T+1移位、流动性过滤（市值>50亿、成交>5000万）、停牌跳过等A股本土化机制，使稀少信号进一步失效。若回测周期短或股票池小，问题更明显。

验证建议：在signal_generator.py中添加debug打印，观察每日条件满足率。

解决方案

采用3步修复框架：修改信号逻辑为加权分数、强化优化器惩罚、执行验证。保留原仓库结构，仅局部替换函数。

步骤1：修改信号生成逻辑（signal_generator.py）

将纯AND替换为加权分数（0-1范围），阈值>0.70触发买入。权重分配参考Barra因子本土化：趋势/BIAS高权重（25%），辅助指标低权重。保留T+1移位与流动性过滤。

# signal_generator.py - 修复版
import pandas as pd
import talib
import numpy as np

def generate_buy_signal(df: pd.DataFrame, params: dict) -> pd.Series:
    # 基础指标计算（同原仓库）
    df['sma60'] = talib.SMA(df['close'], 60)
    df['bias20'] = (df['close'] - talib.SMA(df['close'], 20)) / talib.SMA(df['close'], 20) * 100
    df['rsi14'] = talib.RSI(df['close'], 14)
    df['vol_sma10'] = talib.SMA(df['volume'], 10)
    macd, macdsignal, macdhist = talib.MACD(df['close'])
    df['macd_hist'] = macdhist
    df['is_yang'] = df['close'] > df['open']
    
    # 新增：信号强度分数（0-1）
    score = pd.Series(0.0, index=df.index)
    score += 0.25 * (df['sma60'] > df['sma60'].shift(1)).astype(int)          # 趋势权重25%
    score += 0.25 * np.clip((params['theta_buy'] - df['bias20']) / 10, 0, 1)   # BIAS超跌权重25%
    score += 0.15 * (df['volume'] < params['alpha_vol'] * df['vol_sma10']).astype(int)  # 缩量权重15%
    score += 0.15 * np.clip((params['rsi_thresh'] - df['rsi14']) / 20, 0, 1)   # RSI权重15%
    score += 0.10 * (df['macd_hist'] > 0).astype(int)                         # MACD权重10%
    score += 0.10 * df['is_yang'].astype(int)                                 # 阳线权重10%
    
    # 流动性过滤（A股本土化）
    df['liquidity_ok'] = (df['amount'] > 5e7) & (df['circulating_market_cap'] > 50e8)
    
    buy_signal = (score > 0.70) & df['liquidity_ok']
    return buy_signal.shift(1)  # T+1执行

新买入公式（LaTeX）：
$$
\text{Score}t = 0.25 \cdot \mathbb{I}{\text{SMA60 up}} + 0.25 \cdot \text{clip}\left(\frac{\theta_{\text{buy}} - \text{BIAS20}t}{10},0,1\right) + 0.15 \cdot \mathbb{I}{\text{Vol shrink}} + 0.15 \cdot \text{clip}\left(\frac{\text{rsi_thresh} - \text{RSI14}t}{20},0,1\right) + 0.10 \cdot \mathbb{I}{\text{MACD turn}} + 0.10 \cdot \mathbb{I}{\text{Yang line}}
$$
$$
S{\text{buy},t+1} = \mathbb{I}(\text{Score}_t > 0.70) \land \text{Liquidity OK}
$$

参数建议：θ_buy=-6%~-10%、rsi_thresh=30-35、α_vol=0.5-0.8（遗传算法自动优化）。
