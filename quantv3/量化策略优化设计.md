3. 模块2: 股票过滤 (Dynamic Filtering)

目的： 从池中每日选20-40只高质量股，确保流动性与趋势。

输入： 股票池、日K数据。
输出： 过滤后子集。

过滤规则：

质量：ROE >15%、EPS增长 >20% YoY。
流动性：成交波动 <30%、无连续涨停。
趋势：SMA60向上、Close > SMA200 * 0.9。
政策：北向资金净流入 >0。

关键公式： ROE = Net Profit / Equity (从财务API取)。

4. 模块3: 量化策略 (Signal Generation)

目的： 生成买/卖信号，优化捕捉洗盘回调（BIAS + RSI + MACD）。

输入： 过滤股、日K数据、参数。
输出： DataFrame with 'Signal' (1买/-1卖), 'Target_Shares'。

规则： 如历史所述（买入：SMA60上 + BIAS<-θ + Vol<α*SMA10 + RSI<30 + HIST转正 + 阳线）。

关键公式（LaTeX集成）：

BIAS: ( BIAS_t = \frac{P_t - SMA(P, 20)_t}{SMA(P, 20)_t} \times 100 )
RSI: ( RSI_t = 100 - \frac{100}{1 + RS_t} ), RS = Avg Gain / Avg Loss
MACD HIST: ( HIST_t = (EMA_{12} - EMA_{26}) - EMA_9(MACD) )
Shares: ( Shares = \left\lfloor \frac{Capital \times R}{ATR_{14} \times (1 + Cost)} \right\rfloor \div 100 \times 100 )

伪代码模板：

python
复制
import talib

def generate_signals(df, theta_buy=6.0, rsi_thresh=30, alpha_vol=0.6, R=0.02, capital=1e6, cost=0.0035):
    df['SMA20'] = talib.SMA(df['close'], 20)
    df['BIAS20'] = (df['close'] - df['SMA20']) / df['SMA20'] * 100
    df['SMA60'] = talib.SMA(df['close'], 60)
    df['RSI14'] = talib.RSI(df['close'], 14)
    df['MACD'], df['MACDSignal'], df['HIST'] = talib.MACD(df['close'])
    df['Vol_SMA10'] = talib.SMA(df['volume'], 10)
    df['ATR14'] = talib.ATR(df['high'], df['low'], df['close'], 14)
    
    buy_cond = ((df['SMA60'] > df['SMA60'].shift(5)) &
                (df['BIAS20'] < -theta_buy) &
                (df['volume'] < alpha_vol * df['Vol_SMA10']) &
                (df['RSI14'] < rsi_thresh) &
                (df['HIST'] > df['HIST'].shift(1)) &
                (df['close'] > df['open']))
    
    sell_cond = ((df['BIAS20'] > 12.0) |  # theta_sell示例
                 (df['close'] < df['SMA60'] * 0.95) |
                 ((df['high'] - df['close']) > 0.05 * (df['high'] - df['low'])))
    
    df['Signal'] = 0
    df.loc[buy_cond.shift(1), 'Signal'] = 1  # T+1
    df.loc[sell_cond.shift(1), 'Signal'] = -1
    
    df['Target_Shares'] = 0
    df.loc[df['Signal'] == 1, 'Target_Shares'] = ((capital * R) / df['ATR14'] / (1 + cost)).astype(int) // 100 * 100
    return df
